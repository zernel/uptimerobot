# Contributing

Thank you for your interest in contributing to UptimeRobot Clone!

## Development Setup

1. Fork the repository
2. Clone your fork
3. Install dependencies: `bundle install`
4. Setup database: `bin/rails db:create db:migrate db:seed`
5. Start development: `bin/dev`

## Code Style

- Follow Ruby style guide
- Use RuboCop: `bin/rubocop`
- Write meaningful commit messages

## Pull Requests

1. Create a feature branch: `git checkout -b feature/my-feature`
2. Make your changes
3. Add tests if applicable
4. Run linter: `bin/rubocop`
5. Commit with clear message
6. Push and create PR

## Reporting Issues

- Use GitHub Issues
- Include steps to reproduce
- Include Ruby/Rails versions
- Include error messages
